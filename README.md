# first-pr-practice

A practice repository for learning the GitHub pull request workflow.

## What this repo is for

This repo exists to practice the end-to-end PR process:

1. Clone a repo locally
2. Create a feature branch
3. Make a change and commit it
4. Push and open a pull request
5. Review, merge, and celebrate

## How to use

```bash
git clone https://github.com/Akash1707-alt/first-pr-practice.git
cd first-pr-practice
git checkout -b my-first-branch
# make a change, then:
git add .
git commit -m "my first commit"
git push origin my-first-branch
```

Then open a PR on GitHub — you've got this!
